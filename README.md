# FUTURE_CS_01

## Vulnerability Assessment Report for a Live Website (Read-Only Scope)

### Objective

To analyze a public website for common security weaknesses, classify risks, and provide remediation recommendations using passive security assessment techniques.

### Target Website

https://demo.testfire.net

### Scope

* Public-facing pages only
* Passive scanning
* Security header analysis
* Configuration review

### Tools Used

* Nmap
* OWASP ZAP
* Browser Developer Tools
* Canva

### Findings Summary

A total of 15 findings were identified during the assessment.

Detailed findings covered in the report:

* Absence of Anti-CSRF Tokens (Medium)
* Content Security Policy Header Not Set (Medium)
* Missing Anti-clickjacking Header (Medium)

Additional findings were documented and summarized within the report.
